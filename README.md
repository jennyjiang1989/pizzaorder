# pizzaorder

Jenny's Online Pizza Order 
* Customer can place order with phone number and verification code.
* Administrator can delete order with password.

## Sample
http://pizzaorder.tk/

## Install
brew install mongodb
mongod &

## Start
npm install
npm start
Open browser to access http://localhost:9000
